# **App Name**: Seed Studio

## Core Features:

- Seed Inventory: Manage and track seed inventory with details like name, source, stock level and image.
- Dynamic Logging: Log various gardening activities including planting, harvesting, pest control, watering, pruning, and soil preparation.
- Schedule Management: Create and manage recurring garden tasks with daily, weekly, bi-weekly, or monthly schedules.
- Customizable Tasks: Configure new types of garden tasks and logs, extending the logging system.
- Local Storage Persistence: Store and persist all app data in the browser's local storage.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to evoke a natural and calming gardening vibe.
- Background color: Light beige (#F5F5DC) to create a neutral and unobtrusive backdrop.
- Accent color: Muted brown (#A0522D) for highlights, and actionable elements.
- Body and headline font: 'PT Sans', a modern and readable sans-serif for both headlines and body text.
- Use clear and nature-inspired icons for navigation and actions.
- Clean and responsive layout using Tailwind CSS and ShadCN UI components.
- Subtle transitions and animations for a smooth user experience.
